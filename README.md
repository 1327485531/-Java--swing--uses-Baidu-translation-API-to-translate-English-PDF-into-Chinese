这是一个将英文pdf翻译成中文的小例子，用到了百度翻译api与pdfbox完成。下面是使用过程：
1.首先需要去百度翻译开放平台注册开发者账号，然后找到通用翻译api
 ![](https://github.com/1327485531/-Java--swing--uses-Baidu-translation-API-to-translate-English-PDF-into-Chinese/blob/main/img/%E6%8D%95%E8%8E%B71.JPG)
注意看下产品定价，注册免费的那款。
  ![](https://github.com/1327485531/-Java--swing--uses-Baidu-translation-API-to-translate-English-PDF-into-Chinese/blob/main/img/%E6%8D%95%E8%8E%B72.JPG)

2.注册完成后找到开发者信息，复制下图中的id与key
  ![](https://github.com/1327485531/-Java--swing--uses-Baidu-translation-API-to-translate-English-PDF-into-Chinese/blob/main/img/%E6%8D%95%E8%8E%B73.JPG)

3.启动项目中src目录下pdf目录中的o.java
  ![](https://github.com/1327485531/-Java--swing--uses-Baidu-translation-API-to-translate-English-PDF-into-Chinese/blob/main/img/%E6%8D%95%E8%8E%B74.JPG)

4.在软件界面输入对应的id与key即可进入软件。
  ![](https://github.com/1327485531/-Java--swing--uses-Baidu-translation-API-to-translate-English-PDF-into-Chinese/blob/main/img/%E6%8D%95%E8%8E%B75.JPG)

5.点击识别图片，然后选取需要翻译的英文pdf即可进行翻译，结果会在左面文字窗体显示，翻译过程中不要关闭窗口或进行其他操作。
  ![](https://github.com/1327485531/-Java--swing--uses-Baidu-translation-API-to-translate-English-PDF-into-Chinese/blob/main/img/%E6%8D%95%E8%8E%B76.JPG)

 ![](https://github.com/1327485531/-Java--swing--uses-Baidu-translation-API-to-translate-English-PDF-into-Chinese/blob/main/img/%E6%8D%95%E8%8E%B77.JPG)
